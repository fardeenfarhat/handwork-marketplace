# Assets Directory

This directory contains static assets for the mobile application.

## Required Assets

The following assets need to be added for the app to work properly:

- `icon.png` - App icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `splash.png` - Splash screen image (1284x2778 for iPhone 12 Pro Max)
- `favicon.png` - Web favicon (48x48)

## Asset Guidelines

### App Icon
- Size: 1024x1024 pixels
- Format: PNG
- No transparency
- No rounded corners (handled by platform)

### Splash Screen
- Size: 1284x2778 pixels (iPhone 12 Pro Max)
- Format: PNG
- Should work well with white background
- Keep important content in the center

### Adaptive Icon (Android)
- Size: 1024x1024 pixels
- Format: PNG
- Design should work within a circle, square, and rounded square
- Keep important elements within the safe zone (center 66% of the image)
